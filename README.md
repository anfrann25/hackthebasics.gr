# 🇬🇷 HackTheBasics

Ένα ελληνικό Cybersecurity Journey για αρχάριους.

Στόχος αυτού του project είναι να βοηθήσει νέους ανθρώπους να μάθουν τις βασικές αρχές του Cybersecurity μέσα από θεωρία, εργαστήρια και πρακτικά παραδείγματα.

## Πώς να χρησιμοποιήσεις το Project

Το HackTheBasics έχει σχεδιαστεί ώστε να διαβάζεται μέσα από το documentation interface του project.

1. Κάνε clone το repository.
2. Άνοιξε το αρχείο `index.html` στον browser σου.
3. Χρησιμοποίησε το sidebar για να πλοηγηθείς στα κεφάλαια.
4. Ακολούθησε τη σειρά των ενοτήτων χωρίς να παραλείπεις τα εργαστήρια και τα challenges.

Η προτεινόμενη σειρά είναι:

```text
Introduction
   ↓
Networking
   ↓
Linux
   ↓
Virtual Labs
   ↓
Python Basics
   ↓
Web Security
   ↓
Ethical Hacking
   ↓
CTFs
```

Να δημιουργηθεί ένας οδηγός στα ελληνικά για όσους θέλουν να ξεκινήσουν στον χώρο του Cybersecurity.
