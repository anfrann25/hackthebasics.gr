# DNS

Το DNS (Domain Name System) μετατρέπει ονόματα ιστοσελίδων σε IP διευθύνσεις.

Παράδειγμα:

```text
google.com
```

μετατρέπεται σε:

```text
142.250.x.x
```

---

## Γιατί το χρειαζόμαστε;

Οι άνθρωποι θυμούνται ονόματα.

Οι υπολογιστές επικοινωνούν με IP Addresses.

Το DNS λειτουργεί ως μεταφραστής μεταξύ των δύο.
