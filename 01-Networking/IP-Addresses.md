# IP Addresses

Η IP Address είναι μια μοναδική διεύθυνση που χρησιμοποιείται για την αναγνώριση μιας συσκευής σε ένα δίκτυο.

Παράδειγμα IPv4:

```text
192.168.1.10
```

---

## Public IP

Χρησιμοποιείται στο Internet.

Παράδειγμα:

```text
8.8.8.8
```

---

## Private IP

Χρησιμοποιείται μέσα σε τοπικά δίκτυα.

Παραδείγματα:

```text
192.168.x.x
10.x.x.x
172.16.x.x - 172.31.x.x
```

---

## Πώς βλέπω την IP μου;

Windows:

```powershell
ipconfig
```

Linux:

```bash
ip a
```
