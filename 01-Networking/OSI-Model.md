# OSI Model

Το OSI Model είναι ένα θεωρητικό μοντέλο που περιγράφει τον τρόπο με τον οποίο επικοινωνούν οι συσκευές σε ένα δίκτυο.

Αποτελείται από 7 επίπεδα:

7. Application
6. Presentation
5. Session
4. Transport
3. Network
2. Data Link
1. Physical

---

## Γιατί είναι σημαντικό;

Το OSI Model μας βοηθά να κατανοήσουμε πού ακριβώς συμβαίνει ένα πρόβλημα ή μια επίθεση μέσα σε ένα δίκτυο.

Παράδειγμα:

- DNS → Application Layer
- TCP → Transport Layer
- IP → Network Layer
