# Networking Fundamentals

Πριν ασχοληθούμε με Linux, Web Security ή Ethical Hacking, πρέπει να κατανοήσουμε πώς επικοινωνούν οι συσκευές μεταξύ τους.

Το Networking αποτελεί μία από τις σημαντικότερες βάσεις στο Cybersecurity. Οι περισσότερες επιθέσεις και τεχνικές άμυνας βασίζονται στην κατανόηση του τρόπου λειτουργίας των δικτύων.

---

# Τι είναι ένα Δίκτυο;

Ένα δίκτυο (Network) είναι ένα σύνολο συσκευών που μπορούν να επικοινωνούν μεταξύ τους και να ανταλλάσσουν δεδομένα.

Παραδείγματα συσκευών:

- Υπολογιστές
- Κινητά τηλέφωνα
- Servers
- Εκτυπωτές
- Routers

Παράδειγμα:

Όταν ανοίγεις έναν browser και επισκέπτεσαι μια ιστοσελίδα, η συσκευή σου επικοινωνεί με έναν απομακρυσμένο server μέσω δικτύου.

---

# Βασικές Έννοιες

## IP Address

Κάθε συσκευή σε ένα δίκτυο διαθέτει μια μοναδική διεύθυνση που ονομάζεται IP Address.

Παράδειγμα:

```text
192.168.1.10
```

Η IP λειτουργεί όπως η διεύθυνση ενός σπιτιού.

Χωρίς αυτήν δεν θα γνωρίζαμε πού πρέπει να σταλούν τα δεδομένα.

---

## Port

Οι θύρες (Ports) επιτρέπουν σε πολλές υπηρεσίες να λειτουργούν στην ίδια συσκευή.

Παραδείγματα:

| Port | Υπηρεσία |
|--------|--------|
| 22 | SSH |
| 53 | DNS |
| 80 | HTTP |
| 443 | HTTPS |

Όταν επισκέπτεσαι μια ιστοσελίδα, συνήθως επικοινωνείς με την θύρα 80 ή 443.

---

## Domain Name

Οι άνθρωποι θυμούνται εύκολα ονόματα.

Παράδειγμα:

```text
google.com
```

Οι υπολογιστές όμως χρησιμοποιούν IP διευθύνσεις.

Το DNS μετατρέπει τα Domain Names σε IP Addresses.

---

## DNS

Το DNS (Domain Name System) λειτουργεί σαν τηλεφωνικός κατάλογος του Internet.

Παράδειγμα:

```text
google.com → 142.250.x.x
```

Χωρίς DNS θα έπρεπε να θυμόμαστε αριθμητικές IP διευθύνσεις για κάθε ιστοσελίδα.

---

## HTTP και HTTPS

Το HTTP χρησιμοποιείται για την επικοινωνία μεταξύ browser και web server.

Το HTTPS είναι η ασφαλής έκδοση του HTTP και χρησιμοποιεί κρυπτογράφηση.

Σήμερα οι περισσότερες ιστοσελίδες χρησιμοποιούν HTTPS.

# Γιατί είναι σημαντικό για το Cybersecurity;

Σχεδόν κάθε επίθεση περιλαμβάνει:

- IP Addresses
- Ports
- DNS
- Web Traffic

Αν δεν κατανοείς αυτές τις έννοιες, θα δυσκολευτείς να καταλάβεις εργαλεία όπως:

- Wireshark
- Nmap
- Burp Suite

και γενικά πώς λειτουργούν οι επιθέσεις και οι άμυνες στα δίκτυα.
