# HTTP & HTTPS

Το HTTP χρησιμοποιείται για την επικοινωνία μεταξύ Browser και Web Server.

---

## HTTP

Παράδειγμα:

```text
http://example.com
```

Τα δεδομένα μεταφέρονται χωρίς κρυπτογράφηση.

---

## HTTPS

Παράδειγμα:

```text
https://example.com
```

Τα δεδομένα προστατεύονται μέσω κρυπτογράφησης.

---

## Γιατί είναι σημαντικό;

Το HTTPS προστατεύει:

- Κωδικούς πρόσβασης
- Προσωπικά δεδομένα
- Online συναλλαγές

Σήμερα σχεδόν όλες οι σύγχρονες ιστοσελίδες χρησιμοποιούν HTTPS.
