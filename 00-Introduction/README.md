# Introduction

Καλώς ήρθες στο HackTheBasics.

Πριν ξεκινήσουμε με Networking, Linux και εργαστήρια, είναι σημαντικό να κατανοήσουμε τι είναι το Cybersecurity και γιατί αποτελεί έναν από τους πιο σημαντικούς τομείς της τεχνολογίας σήμερα.

## Τι είναι το Cybersecurity;

Το Cybersecurity (Κυβερνοασφάλεια) είναι η πρακτική προστασίας συστημάτων, δικτύων, εφαρμογών και δεδομένων από μη εξουσιοδοτημένη πρόσβαση, επιθέσεις και κακόβουλες ενέργειες.

Στην καθημερινότητά μας χρησιμοποιούμε:

- Υπολογιστές
- Κινητά τηλέφωνα
- Ιστοσελίδες
- Cloud υπηρεσίες
- Online τραπεζικές υπηρεσίες

Όλα αυτά αποτελούν πιθανούς στόχους επιθέσεων και για αυτό απαιτούνται μέτρα ασφαλείας.

## Γιατί είναι σημαντικό;

Η κυβερνοασφάλεια βοηθά στην προστασία:

- Προσωπικών δεδομένων
- Οικονομικών πληροφοριών
- Επιχειρησιακών συστημάτων
- Κρίσιμων υποδομών

Χωρίς κατάλληλα μέτρα ασφαλείας, ένας επιτιθέμενος μπορεί να αποκτήσει πρόσβαση σε ευαίσθητες πληροφορίες ή να προκαλέσει διακοπή υπηρεσιών.

## Τι θα μάθεις σε αυτό το Journey;

Κατά τη διάρκεια αυτού του project θα γνωρίσεις:

- Βασικές αρχές Δικτύων
- Linux Fundamentals
- Virtual Machines
- Python Basics
- Web Security
- Ethical Hacking
- CTF Challenges

Όλα τα παραδείγματα θα πραγματοποιούνται σε ασφαλή εργαστηριακά περιβάλλοντα.

## Στόχος του Project

Ο στόχος δεν είναι να γίνεις ειδικός από την πρώτη μέρα.

Ο στόχος είναι να αποκτήσεις ισχυρές βάσεις και να κατανοήσεις πώς λειτουργούν τα συστήματα πριν προχωρήσεις σε πιο προχωρημένα θέματα.
