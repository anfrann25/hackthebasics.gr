# Installing VirtualBox

Το **Oracle VirtualBox** είναι ένα δωρεάν εργαλείο που σου επιτρέπει να δημιουργείς και να διαχειρίζεσαι Virtual Machines.

Σε αυτό το κεφάλαιο θα το εγκαταστήσουμε στο σύστημά σου.

---

##  Τι θα χρειαστείς

- Έναν υπολογιστή με Windows, macOS ή Linux
- Σύνδεση στο Internet
- 200–300MB διαθέσιμο χώρο

---

## 🔽 Βήμα 1: Κατέβασμα του VirtualBox

Πήγαινε στη σελίδα: https://www.virtualbox.org/wiki/Downloads

Επίλεξε το installer ανάλογα με το λειτουργικό σου:

- Windows hosts
- macOS hosts
- Linux distributions

---

##  Βήμα 2: Εγκατάσταση

###  Windows

1. Άνοιξε το αρχείο `.exe`
2. Πάτα **Next** σε όλα τα βήματα
3. Επίλεξε **Install**
4. Πάτα **Finish**

###  macOS

1. Άνοιξε το `.dmg`
2. Κάνε διπλό κλικ στο **VirtualBox.pkg**
3. Ακολούθησε τα βήματα εγκατάστασης
4. Αν εμφανιστεί μήνυμα ασφαλείας:
   - *System Settings → Privacy & Security → Allow*

###  Linux (Ubuntu / Debian)

Μπορείς να το εγκαταστήσεις με package manager:

```bash
sudo apt update
sudo apt install virtualbox
