# What is a Virtual Machine?

Μια **Virtual Machine (VM)** είναι ένας εικονικός υπολογιστής που τρέχει μέσα στον πραγματικό σου υπολογιστή.

Σκέψου το σαν έναν δεύτερο υπολογιστή που λειτουργεί ανεξάρτητα, αλλά χρησιμοποιεί τους πόρους (RAM, CPU, αποθηκευτικό χώρο) του δικού σου συστήματος.

---

# 🖥️ Πώς λειτουργεί;

Για να δημιουργήσεις ένα Virtual Machine χρειάζεσαι ένα πρόγραμμα που ονομάζεται **Hypervisor**.

Ο Hypervisor είναι υπεύθυνος για τη δημιουργία και τη διαχείριση των Virtual Machines.

Μερικοί από τους πιο γνωστούς Hypervisors είναι:

- Oracle VirtualBox
- VMware Workstation
- Microsoft Hyper-V

Σε αυτό το course θα χρησιμοποιήσουμε το **Oracle VirtualBox**, καθώς είναι δωρεάν και εύκολο στη χρήση.

---

# 💻 Host και Guest

Όταν δουλεύεις με Virtual Machines θα συναντήσεις δύο βασικούς όρους.

## Host

Ο πραγματικός υπολογιστής σου.

Παράδειγμα:

- Windows 11
- Ubuntu
- macOS

## Guest

Το λειτουργικό σύστημα που εγκαθιστάς μέσα στο Virtual Machine.

Παράδειγμα:

- Kali Linux
- Ubuntu Server
- Windows 10

Έτσι μπορείς να έχεις πολλά διαφορετικά λειτουργικά συστήματα χωρίς να αλλάζεις το βασικό λειτουργικό του υπολογιστή σου.

---

# 🔒 Γιατί χρησιμοποιούμε Virtual Machines στο Cybersecurity;

Οι Virtual Machines αποτελούν βασικό εργαλείο για κάθε επαγγελματία του Cybersecurity.

Μας επιτρέπουν να:

- δοκιμάζουμε εργαλεία με ασφάλεια
- πειραματιζόμαστε χωρίς να επηρεάζουμε τον πραγματικό υπολογιστή
- δημιουργούμε εργαστήρια (Labs)
- δοκιμάζουμε ευπάθειες και επιθέσεις σε ελεγχόμενο περιβάλλον
- επαναφέρουμε εύκολα το σύστημα σε προηγούμενη κατάσταση

---

# ⚠️ Είναι ασφαλές;

Ναι.

Οι Virtual Machines είναι απομονωμένες από το βασικό λειτουργικό σύστημα.

Αυτό σημαίνει ότι μπορείς να κάνεις δοκιμές και πειράματα χωρίς να επηρεάσεις άμεσα τον πραγματικό σου υπολογιστή.

Παρόλα αυτά, καλό είναι να αντιμετωπίζεις κάθε Virtual Machine σαν έναν κανονικό υπολογιστή και να ακολουθείς βασικούς κανόνες ασφαλείας.
