# Virtual Lab

Πριν ξεκινήσεις να ασχολείσαι με εργαλεία Cybersecurity, χρειάζεσαι ένα ασφαλές περιβάλλον όπου μπορείς να πειραματιστείς χωρίς να επηρεάσεις τον πραγματικό σου υπολογιστή.

Σε αυτό το κεφάλαιο θα δημιουργήσεις το πρώτο σου Virtual Lab.

---

## Σε αυτό το κεφάλαιο θα μάθεις

- Τι είναι ένα Virtual Machine
- Γιατί χρησιμοποιούμε Virtual Machines
- Εγκατάσταση του VirtualBox
- Δημιουργία Ubuntu Virtual Machine
- Δημιουργία Kali Linux Virtual Machine
- Τι είναι τα Snapshots

---

## Στόχος

Στο τέλος αυτού του κεφαλαίου θα έχεις ένα πλήρως λειτουργικό εργαστήριο, έτοιμο για τα επόμενα μαθήματα και τα labs του HackTheBasics.
